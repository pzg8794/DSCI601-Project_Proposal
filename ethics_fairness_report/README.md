# DSCI601 Ethics and Fairness Report

This folder contains the submission-ready copy of the one-page DSCI601 ethics/fairness
report.

## Files

- `ethics_fairness_report.tex`: source
- `ethics_fairness_report.pdf`: compiled one-page PDF
- `submission_check.md`: final requirement and readiness check

## Note

The report was developed from the broader ethics foundation kept in the public `IDAI700`
repo, but this copy is the course-deliverable version that belongs with the rest of the
DSCI601 proposal materials.
