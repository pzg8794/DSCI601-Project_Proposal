# Reproducibility Checklist

## Environment
- [ ] Python version specified
- [ ] OS assumptions stated
- [ ] Dependency installation command included
- [ ] Environment file or requirements file referenced

## Data and artifacts
- [ ] Synthetic, inherited, and planned data sources clearly separated
- [ ] Output directory structure documented
- [ ] Artifact manifest location documented
- [ ] Seeds or run counts stated when applicable

## Execution
- [ ] Reviewer-safe quick path documented
- [ ] Stage-by-stage commands documented
- [ ] Quantum local run path documented
- [ ] Optional cloud or long-run path documented

## Validation
- [ ] Test command documented
- [ ] Merge-check command documented
- [ ] Expected outputs listed
- [ ] Figure and table regeneration path listed

## Research transparency
- [ ] Completed vs. planned work labeled explicitly
- [ ] Fairness metrics defined by domain
- [ ] Limitations and validity threats stated
- [ ] Inherited results distinguished from new results
